# Bank Loan Analysis Project
The objective of this project is to extract loan-related data from a banking database, preprocess and analyze it, and generate meaningful insights. The analysis focuses on identifying patterns in good and bad loans, evaluating funded amounts, and assessing overall profitability and losses. A dynamic dashboard is developed in Tableau to present key insights based on defined KPIs.

# Tech Stack & Workflow
Step 1: Requirement Gathering Understand business objectives and reporting needs from stakeholders related to loan performance.

Step 2: Data Extraction Retrieve raw data from the client’s database and import it into MS SQL Server for further processing.

Step 3: Data Processing & Transformation Clean, filter, and structure the dataset based on relevant KPIs using SQL and Excel.

Step 4: Data Cleaning Handle missing values, inconsistencies, and errors to ensure data accuracy and reliability.

Step 5: Data Loading Import the processed dataset into Tableau for visualization.

Step 6: Dashboard Development Design and build an interactive dashboard in Tableau to present insights clearly and effectively.

# Projects Features

- Enables data-driven decision-making by distinguishing between good and bad loans.
- Provides detailed analysis across categories such as loan type, purpose, grade, and verification status.
- Highlights key financial metrics including funded amounts, profits, and losses.
